# GRCB-WEB (Generic Random Cursed Bot - Web Edition)

Esta es una recreación moderna y premium del proyecto GRCB original, llevada a la web con **HTML, TypeScript y CSS**.

## ✨ Características
- **Interfaz Premium**: Diseño oscuro con glassmorphism y animaciones suaves.
- **Lógica de Bot Cursado**: Busca en Reddit y 4chan contenido aleatorio basado en tus preguntas.
- **Modo Imaginación**: Limpia el texto para que parezca una respuesta coherente y original (Ctrl+P para alternar).
- **Traducción Automática**: Traduce resultados de inglés a español en tiempo real.
- **Efecto de Escritura**: Simula al bot escribiendo su respuesta.

## 🚀 Cómo ejecutarlo

1. Abre una terminal en la carpeta `GRCB-WEB`.
2. Instala las dependencias (si no lo has hecho):
   ```bash
   npm install
   ```
3. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```
4. Abre tu navegador en la URL indicada (usualmente `http://localhost:5173`).
